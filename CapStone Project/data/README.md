# Data index

- `sample_input/`: small safe inputs for smoke tests and demonstrations.
- `reports/`: the five authentic RQ PDF reports used for document-level evaluation.
- `benchmark/`: labeled controlled and authentic evaluation CSV files.
- `cache/`: pinned upstream data needed to reproduce SciFact and Fast-DetectGPT experiments.

Generated metrics do not belong here; they are stored under `outputs/evaluation/`.

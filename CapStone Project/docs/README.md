# Documentation index

- `report/`: final LaTeX source, compiled technical report, and report outline.
- `presentation/`: final German demo deck and PDF; reproducibility sources, notes, and media are
  organized in its `source/`, `_support/`, and `assets/` subdirectories.
- `FINAL_RESULTS.md`: concise snapshot of every reportable metric and its limitations.
- `ANNOTATION_GUIDE.md`: labeling guidance for the authentic claim benchmark.

The report and presentation are generated artifacts backed by files in `outputs/evaluation/`.
